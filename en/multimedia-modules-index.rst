Python comes with a small set of modules for dealing with image files
and audio files.



For more extensive support, see ` Python Imaging Library (PIL)
<http://www.pythonware.com/products/pil/>`__ and the ` Snack Sound
Toolkit <http://www.speech.kth.se/snack/>`__, among others.
